# Some Query commands

### Select
``` SELECT column1, column2, ...
    FROM table_name ```

### Where with AND , OR and NOT
``` SELECT column1, column2, ...
    FROM table_name
    WHERE condition1 AND condition2 AND condition3 ...; ```  
``` SELECT column1, column2, ...
    FROM table_name
    WHERE condition1 OR condition2 OR condition3 ...; ```  
``` SELECT column1, column2, ...
    FROM table_name
    WHERE NOT condition; ```
